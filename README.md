# Hotel Booking Website

Project scaffold for a MERN hotel booking system.

## Structure

```text
backend/
	src/
		config/
		controllers/
		docs/
		middleware/
		models/
		routes/
		services/
		utils/
		validators/

frontend/
	src/
		assets/
		components/
		context/
		hooks/
		layouts/
		pages/
		routes/
		services/
		utils/

docs/
	postman/
	swagger/
```

## Feature Areas

- Browse hotels, rooms, and amenities
- Search and filter by location, dates, price range, and amenities
- Booking flow with availability updates
- Secure auth, authorization, and rate limiting
- API validation via Postman and Swagger
- Email confirmations and booking history
- Promotions such as discount codes and loyalty rewards
